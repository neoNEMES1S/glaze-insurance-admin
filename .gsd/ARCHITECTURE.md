## Data Flow

1. User interacts with Next.js App Router pages (`src/app`).
2. Components dispatch actions or execute React Query hooks to fetch/mutate data.
3. React Query invokes API calls through `src/lib/api.ts`.
4. `api.ts` adds authorization headers and communicates with the external backend (`NEXT_PUBLIC_API_URL` or `http://localhost:8000`).
5. Backend responds, components update based on React Query / Zustand state changes.
